# PySpark-DataFrame-Practice
This repository demonstrates basic PySpark operations like DataFrame creation, nested structures, select statements, column operations, filtering, and array functions using simple examples.

## Topics Covered

- Creating DataFrames with schema
- `show()` with `truncate=True` and `truncate=False`
- Selecting columns using different methods
- Selecting nested struct columns
- Using `df.columns` and column slicing
- DataFrame immutability (creating new DataFrames)
- Adding constant column using `lit()`
- Casting column types using `col().cast()`
- Creating derived columns
- Filtering DataFrames (`==`, `!=`, `isin`, `startswith`, `contains`)
- Working with Array columns
- Using `array_contains()`
- Pattern matching using `like`, `rlike`, and `ilike`

## Requirements

- Python
- Apache Spark
- PySpark
